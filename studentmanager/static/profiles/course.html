CourseManager API: Course Profile

In the API, a Course resource represents the information of a course offered by the University. It holds the following attributes:
  • title: the course's title. It is a string limited to 255 characters in length.
  • teacher: the name of the course's teacher. It is a string limited to 255 characters in length.
  • code: the course's unique code. It is a string limited to 12 characters in length.
  • ects: the amount of ECTS the course grants upon completion. It is an integer value.
All attributes are mandatory when adding or modyfing a course. A complete JSON schema for courses can be obtained from the Course.json_schema method, or by the hypermedia controls related to adding or modyfing courses.

Endpoints

All courses are stored in the CourseCollection resource (/api/courses/). This collection allows to add new courses to the database.

Individual courses can be accessed from the CourseItem resource (/api/courses/<course_id>/). This resource allows editing and deletion of an individual course.
It also allows access to the course's assessment collection (/api/courses/<course_id>/assessments/).

Relevant link relations

  • self: carried by all Collections and Items returned by the API. In particular, each course returned by the CourseCollection will carry it.
  • profile: points to this page.
  • collection: points from each Item resource to the respective Collection. (i.e. CourseItem resources will point to the CourseCollection).
  • studman:students-all, studman:assessments-all: point to the other two main Collections in our API structure: StudentCollection and AssessmentCollection.
  • studman:add-course: carried by CourseCollection, pointing to itself specifying the POST method and the required Course json schema.
  • edit, studman:delete: carried by CourseItem, pointing to itself with PUT and DELETE methods respectively.
  • studman:course: carried by the CourseAssessmentCollection resource, points to the relevant Course.
  • studman:course-assessments: inverse of the previous control, carried by a CourseItem, points to the relevant CourseAssessmentCollection.
